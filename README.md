# 🧑‍💼 Employee Leave Management System

## 📖 Introduction
This project is a simple **Spring MVC web application** designed to demonstrate the core flow of authentication and form handling in a real-world scenario. It simulates how an organization manages employee access and leave requests through a structured web interface.

---

## 📌 Description
The application starts with a secure login page where employees must enter valid credentials to proceed. Upon successful authentication, users are redirected to a leave application form where they can submit their leave details. Once the form is submitted, the system processes the input and displays a confirmation message on the leave approval page.

If incorrect login credentials are entered, the system prevents access and redirects the user back to the login page with an appropriate error message. This ensures basic access control and improves user experience by providing immediate feedback.

The project highlights key concepts such as:
- Request handling using Spring MVC controllers  
- Form submission and data binding  
- Page navigation and redirection  
- Basic validation and user feedback  

Overall, this project serves as a foundational implementation for building more advanced enterprise-level applications with authentication and workflow management.
